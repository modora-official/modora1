## âš™ï¸ Admin Panel Guide

Untuk mengupload aplikasi baru, gunakan file `admin.html`:

1.  Buka `admin.html`.
2.  Pilih Tab **APPS** atau **GAMES**.
3.  Masukkan **Nama Aplikasi** dan **Link Download**.
4.  Upload **Thumbnail** (Otomatis dikompres & di-resize).
5.  Klik **PROSES DATA**.
6.  Sistem akan otomatis:
    * Mengompres gambar.
    * Membuat folder baru di repo.
    * Mengupload file `index.html` (halaman detail) dan gambar.
    * Mengupdate `index.html` utama (Root) agar aplikasi muncul di list.
    * Mengirim notifikasi ke Telegram Channel.
